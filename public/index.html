KickAxe Inventory System

Index Page

Home Page

This page shows options to Log-In, Create Account, or Browse the database.

User Login

This page prompts the user for their blacksmith username/password and has options to go back to the Home Page or Create an Account.

Create Account

This page effectively “inserts” a blacksmith into the database by allowing them to create a blacksmith account.

Search Page

If not logged in, this page gives the user options to search Weapons, Enchantments, Materials, or Blacksmiths.

If logged in, this page gives the user options to search Weapons, Enchantments, Materials, or Blacksmiths as well as giving them the option to view their own Profile.

User Profile Page

This page lists weapons made (can link to specific weapon page), materials owned (can link to specific material page), location, and current/past mentors/mentees the blacksmith has had.

Profile Editor

This page allows logged-in blacksmiths to add new mentor/mentee training relationships, add/remove weapons, add/remove owned materials, and/or change their experience level.

Specific Weapon Page

This page shows the stats, materials (link to specific material page), and enchantments (link to specific enchantment page) of a specific weapon.

Specific Material Page

This page shows the stats, and current weapons with this material (links to specific weapon page) of a specific material.

Specific Enchantment Page

This page shows stats, experience requirement, material requirements (links to specific material page), and current weapons with this enchantment (links to specific weapon page) of a specific enchantment.
